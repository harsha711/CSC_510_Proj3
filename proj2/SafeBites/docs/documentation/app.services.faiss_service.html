Build a FAISS vector store from dish data JSON. Args: json_path (str): Path to the dish JSON data. Raises: FileNotFoundError: If JSON file does not exist. GenericException: If index creation fails.

def extract_query_intent(query): ¶

Extract positive and negative intents from a user query. Args: query (str): User natural language query. Returns: QueryIntents: Positive and negative intents. Notes: Falls back to {"positive": [query], "negative": []} if LLM fails.

def refine_with_centroid(dish_hits, positive_intents, dish_embeddings): ¶

Refine retrieved dishes by comparing centroid similarity of positive intents. Args: dish_hits (List[DishHit]): Initial dish hits. positive_intents (List[str]): Positive intents extracted from query. dish_embeddings (Dict[str, np.ndarray]): Precomputed embeddings of dishes. Returns: List[DishHit]: Refined and sorted dish hits.
